# **App Name**: Karma Ledger

## Core Features:

- Good Deed Submission: Submit a description of a good deed, the type of kindness, and an optional photo, storing the data on-chain or via IPFS with a contract reference.
- Verification Flow: Allow users to verify deeds via on-chain signatures, triggering verification once a threshold is reached.
- Karma Shard Minting: Mint ERC-1155 Karma Shards upon deed verification, with different shard types for various kindness categories.
- Karma Aura Fusion: Fuse Karma Shards into dynamic ERC-721 Karma Auras, updating aura metadata as reputation grows.
- Aura Upgrade: Automatically upgrade the Aura level based on verified deeds, visualized using on-chain SVGs with gradients and soft glows.
- Home Feed: Display an infinite scroll of recent deeds, encouraging community engagement and awareness.
- Karma Index Tool: Employ an LLM to provide helpful suggestions for generating a ‘Karma Index’ that shows the overall positive contributions on the platform, adapting displayed insights and information based on verified user actions. The LLM functions as a helpful tool to maintain relevant community feedback.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA) to evoke a sense of calm and spirituality.
- Background color: Very pale lavender (#F5F5FF) to provide a soft, unobtrusive backdrop.
- Accent color: Soft emerald (#C1E1C1) to symbolize growth and harmony, complementing the lavender tones.
- Body and headline font: 'PT Sans' (sans-serif) for a modern yet warm feel.
- Use gentle, glowing icons that reflect kindness and community.
- Employ glass panels with a warm blur effect to create a soft-futuristic aesthetic.
- Incorporate floating glowing particles in the background and a calming Karma Wave animation on deed verification.
